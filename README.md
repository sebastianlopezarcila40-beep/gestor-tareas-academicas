# Software de Tareas Académicas

Proyecto realizado en Python con Flask, HTML, CSS y SQLite.

## Funciones incluidas

### Primera versión
- Inicio
- Lista de tareas
- Crear tarea
- Editar tarea
- Materia

### Segunda versión
- Agregar tarea
- Completar tarea
- Eliminar tarea

### Tercera versión
- Login
- Filtros
- Fecha
- Diseño visual
- Estadísticas

## Cómo ejecutar el proyecto

1. Abrir la carpeta en Visual Studio Code.
2. Abrir una terminal dentro de la carpeta.
3. Instalar Flask:

```bash
python -m pip install flask
```

4. Ejecutar el sistema:

```bash
python app.py
```

5. Abrir en el navegador:

```text
http://127.0.0.1:5000
```
